1. What term describes the number of unique values in a specific dataset?

2. What term is used to describe the actual set of data stored in a database table?
   - Explanation: A relation instance is the concrete collection of data stored within a database table, also known as the relation or extension.

3. In the context of databases, what does a tuple represent?
   - Explanation: A tuple refers to a single row or record within a database table, containing data fields corresponding to the table's attributes.

4. Which protocol is used to send error messages and operational information in IP networks?

   - Explanation: A virus is a type of malware that replicates itself and spreads to other computers by attaching to host files or through network transmissions.

7. What term describes a unit of data transmitted independently over a network without establishing a connection?

8. In networking, what is a pre-defined logical communication path between sender and receiver stations called?

1. **Cardinality**: Cardinality in databases refers to the uniqueness of data values contained in a column or dataset. It measures the number of distinct values present and is crucial for database optimization, indexing strategies, and data analysis. Understanding cardinality helps database administrators and analysts assess data quality, identify potential anomalies or duplicates, and optimize query performance by choosing appropriate indexing strategies based on the distribution of unique values.

2. **Relation Instance**: In the context of relational databases, a relation instance refers to the set of rows (tuples) currently stored in a database table. It represents the actual data content at a given moment, distinct from the table schema (relation schema) which defines the structure and attributes of the table. The distinction between relation schema (structure) and instance (data) is fundamental for database operations such as data retrieval, insertion, deletion, and updating. It also impacts the management of database transactions and the enforcement of data integrity constraints.

3. **Tuple**: A tuple in database management systems (DBMS) represents a single record or row within a relational database table. It consists of a set of attributes (columns) that describe specific characteristics or properties of the entity being represented. Tuples are fundamental units of data storage and retrieval in relational databases, enabling structured and organized storage of information. DBMS operations, such as querying with SQL (Structured Query Language), manipulate tuples to retrieve, update, or delete specific data records based on defined criteria.

4. **ICMP**: The Internet Control Message Protocol (ICMP) is an integral part of the Internet Protocol (IP) suite and is used for diagnostic and error reporting purposes in IP networks. ICMP allows network devices, such as routers and hosts, to send error messages and operational information to indicate network conditions or problems. Common uses of ICMP include reporting unreachable destinations, facilitating the Path MTU (Maximum Transmission Unit) discovery process, and testing network connectivity with tools like ping. ICMP messages are crucial for network troubleshooting, monitoring, and maintenance.

5. **Virus**: A computer virus is a malicious software program designed to replicate itself and spread to other computers or networks. Viruses typically attach themselves to legitimate executable files or disguise themselves as harmless programs or email attachments to trick users into executing them. Once activated, viruses can perform harmful actions such as deleting or corrupting files, stealing sensitive information (such as passwords or financial data), and disrupting normal system operations. Effective antivirus software and proactive security measures are essential for detecting, preventing, and mitigating virus infections.

6. **SSH**: Secure Shell (SSH) is a cryptographic network protocol used for secure remote access to computers and servers over an unsecured network. SSH provides strong encryption and authentication mechanisms to protect data integrity and confidentiality during remote login sessions, file transfers (SCP and SFTP), and command executions. Unlike older protocols like Telnet, which transmit data in plain text, SSH encrypts all communications, including passwords and commands, thereby significantly reducing the risk of eavesdropping, man-in-the-middle attacks, and unauthorized access.

7. **Datagram**: In computer networking, a datagram refers to an independent, self-contained packet of data transmitted over a network. Unlike connection-oriented communication (such as TCP/IP), which establishes a dedicated path or circuit before data transmission, datagram-based communication (such as UDP/IP) sends packets independently without prior setup. Each datagram includes necessary routing information (such as source and destination addresses) and can take different routes to reach its destination, making it suitable for applications requiring fast transmission and minimal overhead, such as real-time multimedia streaming or online gaming.

8. **Virtual Circuit**: A virtual circuit is a logical communication path established between sender and receiver stations in packet-switched networks before actual data transmission occurs. Unlike traditional circuit-switched networks, which establish dedicated physical connections for data transmission, virtual circuits use packet-switching technology to create temporary, logical paths through shared network infrastructure. Virtual circuits ensure reliable and ordered delivery of data packets by assigning sequence numbers and maintaining connection state information, enhancing network efficiency and reducing transmission delays. They are commonly used in networking protocols like Frame Relay and ATM (Asynchronous Transfer Mode) for optimized data transmission across wide area networks (WANs).

9. **AJAX**: Asynchronous JavaScript and XML (AJAX) is a web development technique used to create interactive and dynamic user interfaces by asynchronously exchanging data with a web server in the background. AJAX enables web pages to update specific content sections dynamically without reloading the entire page, providing a smoother and more responsive user experience. By leveraging JavaScript for client-side scripting and XML (or other data formats like JSON) for data interchange, AJAX allows applications to fetch and display updated information in real-time, facilitate user interactions (such as form submissions and content updates), and enhance overall usability and performance.

10. **Trojan Horse**: A Trojan horse, or simply Trojan, is a type of malicious software (malware) that masquerades as legitimate software or files to deceive users into executing it. Unlike viruses and worms, Trojans do not self-replicate but rely on social engineering tactics (such as email attachments, software downloads, or fake websites) to trick users into installing and running them. Once activated, Trojans can perform various malicious actions, such as stealing sensitive information (including login credentials and financial data), installing backdoors for remote access by attackers, and damaging system files or software. Effective cybersecurity practices, including cautious browsing habits and robust antivirus protection, are essential for detecting and preventing Trojan infections.

11. **ICMP**: The Internet Control Message Protocol (ICMP) facilitates communication between network devices by transmitting error messages and operational information. It plays a crucial role in network diagnostics, helping administrators identify and resolve network issues promptly.
